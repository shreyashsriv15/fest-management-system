Event & Participant Management System (C++)

Project Description

A console-based Event & Participant Management System built using C++ and Object-Oriented Programming (OOP) principles.
This application enables efficient management of events, participants, and administrative operations through a structured menu-driven interface.

Key Features
1) Authentication System
- Admin account creation
- Secure login validation
- Program termination on invalid credentials

2) Event Management
- Add new events
- Delete existing events
- Update schedule (date, time, venue)
- View all events
- Declare winners
- Generate event reports
- Search events by name/date/organiser
- View organiser details
3) Participant Management
- Set maximum participant limit
- Add participants to events
- Delete participants

  
Concepts & Skills Demonstrated

This project showcases:

- Object-Oriented Programming (OOP)
- Classes & Objects
- Inheritance
- Encapsulation
- Modular Programming
- Menu-driven Application Design
- Data Handling using Arrays
- User Input Handling
